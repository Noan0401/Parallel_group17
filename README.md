# Parallel_group17

モンテカルロ法の円周率推定計算を並列処理で実行する

 
# Requirement
  
* Python 3.7.1
 
# Usage
```
python Main.py
```

# Note
 
Main.pyの以下の関数の値を変えれば違う実行結果も得られます

Para(n):点のプロット結果なしの関数

n:タスク分割数 

Para2(n,N)  :点のプロット結果ありの関数

n           : タスク分割数 

N           :実行回数

```
Para.Para(n)

Para.Para2(n,N)
```
